### How to Get Listed

If you contribute to Modulo, add yourself here as part of your Pull Request:

```markdown
- **[Your Name](https://github.com/your-username)** — Brief description of your contribution
```

All contributions are valued and welcome, including:

- 🐛 Bug fixes
- ✨ New features
- 📖 Documentation improvements
- 🧪 Test coverage
- 🔧 Infrastructure and tooling
- 🔒 Security fixes
- 🎨 UI/UX enhancements
- ⚡ Performance improvements
